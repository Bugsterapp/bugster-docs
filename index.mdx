Bugster CLI generates and runs AI-powered end-to-end tests for your web applications. It analyzes your codebase, creates realistic test scenarios, and executes them automatically.

## What is Bugster?

Bugster transforms manual testing into automated test cases by understanding your application structure and generating intelligent test specifications that validate user workflows.

<CardGroup cols={2}>
  <Card title="AI-Powered Analysis" icon="brain">
    Automatically analyzes your codebase to understand pages, components, and user flows
  </Card>
  <Card title="Browser Automation" icon="window-restore">
    Executes tests using real browsers with Playwright automation
  </Card>
  <Card title="Team Collaboration" icon="users">
    Share tests across your team with built-in synchronization
  </Card>
  <Card title="CI/CD Integration" icon="workflow">
    Runs automatically on pull requests and deployments
  </Card>
</CardGroup>

## How It Works

<Steps>
  <Step title="Initialize">
    Run `bugster init` to configure your project and authenticate
  </Step>
  <Step title="Generate">
    Run `bugster generate` to create AI-powered test specifications
  </Step>
  <Step title="Execute">
    Run `bugster run` to execute tests with browser automation
  </Step>
</Steps>
