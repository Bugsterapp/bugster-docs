## Overview

**Purpose**: Configure or update your Bugster API key for authentication.

**When to use**:
- When switching between organizations/teams
- Setting up Bugster for the first time
- If your API key expires or gets rotated
- When API key authentication fails

**Important notes**:
- API keys start with `bugster_` prefix
- Stored securely in `~/.bugsterrc` 
- Can also use `BUGSTER_CLI_API_KEY` environment variable
- Required for all Bugster operations

## CLI Reference

| Command | Description | Example |
|---------|-------------|---------|
| `bugster auth` | Interactive API key setup | `bugster auth` |
| `bugster auth --api-key <key>` | Set API key directly | `bugster auth --api-key bugster_org_abc123...` |
| `bugster auth --clear` | Remove existing API key | `bugster auth --clear` |

### Flags

| Flag | Description | Example |
|------|-------------|---------|
| `--api-key` | Bugster API key to set (starts with 'bugster_') | `--api-key bugster_org_abc123...` |
| `--clear` | Clear the existing API key | `--clear` |
