## Prerequisites

<Warning>
You must have the [Bugster GitHub App](https://github.com/apps/bugster-dev) installed and connected to your repository before using this integration.
</Warning>

Visit your [Bugster Dashboard](https://app.bugster.dev) → Project Settings → GitHub Integration to install the GitHub App if you haven't already.

## Automatic Integration

<Check>
The Railway integration activates automatically when you connect your GitHub repository. No additional configuration is required.
</Check>

Railway integration works seamlessly with your existing GitHub connection to automatically run tests on your deployments.
